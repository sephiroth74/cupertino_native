## 0.2.0-dev

* Added `CNSplitView` and `CNSplitViewController` for desktop two-pane layouts.
* Added Split View demo page with practical controls and mixed-content validation.
* Added divider drag snapping with configurable hysteresis (`snapThreshold`, `snapReleaseThreshold`).
* Added optional desktop UX features for Split View (keyboard shortcuts, divider double-click action, accessibility semantics).
* Added optional macOS-specific divider visual enhancements while preserving Flutter-first behavior.
* Added dedicated widget tests for Split View sizing, collapse/expand restore, snapping, and resize metrics.
